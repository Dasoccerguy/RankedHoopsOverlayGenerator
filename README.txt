This runs best on Python 3.4 (3.5 or 3.6 will work)

You will need to install PIL (Python Imaging Library), so look that up and figure out how to install it.

Be sure to set UI scale to 90% when casting.

To run the program, either double click run.bat, or navigate here in command line and do "start pythonw -m OverlayGenerator"
